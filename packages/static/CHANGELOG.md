# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Semantic Versioning](https://semver.org/lang/pt-BR/).

## [1.3.0] - 18/09/2025

### Adicionado
- Suporte completo para Hot Module Replacement (HMR) com logging
- Sistema de gerenciamento de coleções para arquivos Markdown
- Logger específico para HMR com controle de verbosidade
- Manager de HMR para recarregamento automático de coleções
- Refatoração de arquivos para padrão kebab-case

### Melhorado
- Plugin Vite aprimorado com suporte a HMR nativo
- Sistema de build com melhor organização de arquivos
- Configuração do TypeScript atualizada para compatibilidade
- Padronização de nomenclatura de arquivos e módulos
- Otimização do processo de watch de arquivos Markdown

### Corrigido
- Caminhos de export atualizados para usar kebab-case consistentemente
- Melhoria na detecção de mudanças em arquivos de coleção

## [1.2.0] - Março/2025

### Adicionado
- Configuração Jest e scripts de teste
- Suporte básico para sites estáticos
